# Account-App


一款簡單直覺的記帳應用，幫助你輕鬆管理收支，掌握財務狀況。

🔗 **線上體驗**：[Account-App](https://yuzen9622.github.io/Account-App/)

## 功能特色
- 📊 **記錄收支**：快速新增支出與收入，分類清晰。
- 📅 **時間篩選**：按日、週、月查看財務狀況。
- 📈 **統計圖表**：直觀分析你的消費習慣。
- 🔒 **安全性**：個人帳戶數據加密存儲。

## 技術架構
- **前端**：React (使用 CRA 建置)
- **後端**：Node.js (Express.js API)
- **資料庫**：MongoDB
- **驗證系統**：JWT 驗證機制
- **伺服器**：Github
